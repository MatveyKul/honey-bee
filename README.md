# 🍯 Honey Bee - Магазин натурального мёда

Адаптивный лендинг для магазина натурального мёда. Проект демонстрирует навыки вёрстки, работы с Flexbox, Grid, медиазапросами и создания современного дизайна.

## ✨ Особенности

- ✅ Полностью адаптивная вёрстка (мобильные, планшеты, десктоп)
- ✅ Семантическая структура HTML
- ✅ Flexbox и Grid для построения сеток
- ✅ Hover-эффекты и плавные анимации
- ✅ Кастомные шрифты через @font-face
- ✅ Форма подписки с валидацией
- ✅ Русскоязычный интерфейс

## 🛠️ Технологии

| Технология | Назначение |
|------------|------------|
| HTML5 | Структура страницы |
| CSS3 | Стилизация, анимации, адаптив |
| Flexbox | Построение гибких макетов |
| Grid Layout | Сетки для товаров и галереи |
| Media Queries | Адаптив под разные устройства |

## 📱 Адаптивность

| Устройство | Ширина экрана | Особенности |
|------------|---------------|-------------|
| Мобильные | до 768px | Вертикальное расположение блоков, уменьшенные отступы |
| Планшеты | 768px - 1024px | Оптимизированные размеры текста |
| Десктоп | от 1024px | Полная версия с горизонтальным расположением |
